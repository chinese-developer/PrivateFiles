# 切换分支并推送到服务器
1）查看当前要同步的分支 git branch
2）保证当前分支工作目标是干净的 git status
3）新建本地分支 git checkout -b feature_sdk
4）查看现在分支的状态 git branch （星号(*)表示当前所在分支）
5）把新建的本地分支push到远程服务器 git push origin feature_sdk:feature_sdk (要推送的<本地分支名>:推送到<远程分支名>)
6）查看所有分支，如果存在新建的分支，则成功 git branch -a
7）删除远程分支 git push origin :feature_sdk （推送一个空的分支到远程分支，其实就是删除）
